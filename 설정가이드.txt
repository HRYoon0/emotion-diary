================================================
  감정일기 - 설정 가이드
================================================

📌 전체 구조:
  - index.html      → 학생용 감정 기록 페이지
  - teacher.html    → 교사용 대시보드
  - apps-script/Code.gs → Google Sheets 연동 코드


================================================
  1단계: Google Sheets 준비
================================================

1. Google Drive에서 새 스프레드시트를 만듭니다.
   - 이름 예시: "5학년 감정일기"

2. 상단 메뉴에서 [확장 프로그램] > [Apps Script]를 클릭합니다.

3. 기존 코드를 모두 지우고, apps-script/Code.gs 파일의
   내용을 전체 복사하여 붙여넣습니다.

4. 저장 (Ctrl+S) 합니다.

5. Apps Script 상단 함수 선택에서 "setupSheet"를 선택하고
   ▶ 실행 버튼을 클릭합니다.
   - 처음 실행 시 권한 허용이 필요합니다.
   - "고급" > "프로젝트명(안전하지 않음)(으)로 이동"을 클릭합니다.

6. 시트가 자동으로 생성됩니다:
   - "감정기록" 시트: 학생들의 감정 데이터가 저장됨
   - "학생목록" 시트: 학생 정보 (반/번호/이름/비밀번호)
   - "교사설정" 시트: 교사 비밀번호


================================================
  2단계: 학생 목록 입력
================================================

"학생목록" 시트에 실제 학생 정보를 입력합니다.

  반 | 번호 | 이름   | 비밀번호
  1  | 1    | 김민준 | 1234
  1  | 2    | 이서연 | 1234
  ...

※ 비밀번호는 간단한 숫자로 설정 (초등학생용)
※ 기본 비밀번호를 모두 동일하게 설정 후, 학생이 기억하기 편한 것으로 변경 가능


================================================
  3단계: 교사 비밀번호 설정
================================================

"교사설정" 시트에서 교사 비밀번호를 변경합니다.

  설정           | 값
  교사비밀번호   | teacher2024  ← 원하는 비밀번호로 변경


================================================
  4단계: 웹 앱으로 배포
================================================

1. Apps Script에서 [배포] > [새 배포]를 클릭합니다.

2. 톱니바퀴 아이콘(⚙️) 옆 "유형 선택"에서 "웹 앱"을 선택합니다.

3. 설정:
   - 설명: "감정일기 API"
   - 실행 계정: "나"
   - 액세스 권한: "모든 사용자"

4. [배포] 클릭 후, 나타나는 URL을 복사합니다.
   (예: https://script.google.com/macros/s/ABC.../exec)


================================================
  5단계: 웹페이지 열기
================================================

1. index.html 파일을 브라우저에서 엽니다.
   (더블클릭 또는 브라우저로 드래그)

2. 첫 화면에서 4단계에서 복사한 URL을 붙여넣고 저장합니다.

3. 학생 로그인 테스트:
   - 반: 1
   - 번호: 1
   - 비밀번호: 1234 (학생목록에 설정한 비밀번호)

4. 교사 대시보드는 teacher.html에서 확인합니다.


================================================
  사용 팁
================================================

▶ 학생 사용법:
  1. 로그인
  2. 교시 선택
  3. 감정 이모지 선택
  4. 감정 강도 조절 (1~5)
  5. 메모 작성 (선택)
  6. "감정 기록하기" 클릭
  7. "내 기록" 탭에서 날짜별 확인

▶ 교사 사용법:
  1. teacher.html에서 로그인
  2. 반/날짜/교시 필터 설정 후 조회
  3. 감정 분포 차트 확인
  4. 학생별 상세 기록 확인
  5. Google Sheets에서 직접 데이터 확인/분석도 가능

▶ 매 시간(교시) 활용 방법:
  - 수업 시작 전 1분간 감정 기록 시간 부여
  - 하루 끝에 "내 기록" 탭에서 감정 변화 확인
  - 교사가 대시보드에서 반 전체 감정 흐름 파악

▶ Google Sheets 직접 활용:
  - 감정기록 시트에서 필터, 정렬, 차트 등 활용 가능
  - 데이터를 복사하여 다른 분석 도구에서도 사용 가능


================================================
  문제 해결
================================================

Q: "서버에 연결할 수 없습니다" 오류
A: API URL이 올바른지 확인. 배포를 다시 해야 할 수 있음.

Q: 로그인이 안 됩니다
A: 학생목록 시트에 반/번호/비밀번호가 정확히 입력되어 있는지 확인.

Q: 코드를 수정한 후 반영이 안 됩니다
A: Apps Script에서 [배포] > [배포 관리] > "새 버전"으로
   다시 배포해야 수정사항이 반영됩니다.

Q: 여러 반을 운영하고 싶습니다
A: 학생목록 시트에 각 반 학생을 모두 입력하면 됩니다.
   교사 대시보드에서 반별 필터링이 가능합니다.
